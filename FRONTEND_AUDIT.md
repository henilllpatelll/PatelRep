| 28 | `tasks/page.tsx` | Comment input in `TaskDetailDrawer` has no label | **High** | `<input placeholder="Add a comment..."/>` — placeholder disappears on focus, leaving blind users with no context. | Add `aria-label="Add a comment"` to the input. |
| 29 | `tasks/page.tsx` | `TaskDetailDrawer` `w-96` overflows on 375px mobile | **Medium** | `w-96` = 384px. On a 375px phone this clips. | 30 | `tasks/page.tsx` | No success feedback after a comment is posted | **Low** | Input clears on success but nothing else happens. | 32 | `Sidebar.tsx` | `<aside>` has no `aria-label` | **High** | Screen readers announce a generic "complementary" landmark. Users can't identify this as primary navigation. | Add `aria-label="Main navigation"` to the `<aside>` element. |

| 37 | `reports/page.tsx` | `text-yellow-600` used for mid-range SLA — fails WCAG 1.4.3 contrast | **High** | `yellow-600` (#CA8A04) on white has a contrast ratio of ~2.9:1, far below the 4.5:1 minimum for normal text. ## Priority Fix Order

These are the issues that break real workflows or block users with assistive technology first:

1. **Fix `STATUS_BG` → inline style bug** in `RoomStatusBoard` — status filter dots are currently invisible (#16)
2. **`htmlFor`/`id` label pairing** — every modal form field is inaccessible (#23)
3. **`role="dialog" aria-modal` + focus trap + Escape key** on all modals (#38, #39)
4. **`aria-live`** for AI chat message stream (#8)
5. **`aria-current="page"`** and `aria-label="Main navigation"` on Sidebar (#32, #33)
6. **`role="button"` + keyboard handlers** on `TaskCard` and `HousekeeperRoomItem` (#14, #24)
7. **`aria-label`** on chat inputs, send buttons, header dropdown, AI panel close button (#4, #7, #34)
8. **`aria-pressed`** on all filter chip buttons (#18)
9. **Fix `SlaIndicator` stale clock** — `useState` → `setInterval` (#26)
10. **Fix `EditRequestModal` losing existing assignment** on every save (#20)
11. **Remove frosted glass from task cards and `HousekeeperBar`** — cerebrum rule violation (#12, #25)
12. **Fix contrast** — replace `text-yellow-600` → `text-amber-700` in Reports (#37)
13. **Fix AI Copilot page height calc** — `4rem` → `3.5rem` (#9)
14. **Confirm button tap target** — `py-2` → `py-3` on all Copilot confirm buttons (#11)
15. **Date nav button tap targets** — `px-2.5 py-1` → `px-3 py-2` (#15)
