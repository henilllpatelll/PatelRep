
    STEP 2 — Add google-services.json (required for Android EAS build):
    - Go to: Firebase Console -> Project Settings -> Your Apps -> Android app (com.patelrep.app)

    - Click "Download google-services.json"
    - Place it at: apps/mobile/google-services.json
    - Commit the file (it contains only public identifiers, safe to commit)

    STEP 3 — Upload FCM V1 service account for push notifications (optional for first build, required for push):
    - Go to: Firebase Console -> Project Settings -> Service accounts -> Generate new private key
    - Save the JSON file
    - Run: `eas credentials` -> Select Android -> Add FCM V1 service account key -> upload the JSON

    STEP 4 (iOS, can defer if Apple Developer not enrolled):
    - If enrolled: `eas build --platform ios --profile production`
    - If not enrolled yet: skip iOS — Android APK alone satisfies the pilot's immediate need

    STEP 5 — Trigger Android APK build to verify everything works:
    ```
    cd apps/mobile
    eas build --platform android --profile preview
    ```
    This will take 5-15 minutes in the EAS cloud. Download the APK and sideload it to an Android device to confirm it installs.
